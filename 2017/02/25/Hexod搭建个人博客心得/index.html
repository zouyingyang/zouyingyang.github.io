Hexod搭建个人博客心得

发表于 2017-02-25 | 分类于 hexo |

最近突然想建个个人博客,所以花了一个晚上的时间利用hexo,github搭建起来,跟大家说一下我在搭建过程中的一些心得.

配置环境

安装Node(晚上有很多node安装教程)
安装Git(Xcode自带git)

全局安装

安装Hexo
在终端中输入 $ sudo npm install -g hexo

blog项目下

新建blog项目文件
cd到你的blog项目下面 $ cd blog
安装依赖blog$ npm install
初始化blogblog$ hexo init
生成静态页面blog$ hexo generate
终端命令显示如下命令就表示生成静态页面成功

本地启动$ blog$ hexo server

终端命令显示如下命令表示本启动成功
INFO Start processing INFO Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

浏览器输入http://localhost:4000
至此本地已经调试成功

###Github配置
建立Repository
（需要注意的是仓库名必须用 username.github.io。比如我的用户名为 abc 则仓库名称为 abc.github.io）
打开blog项目下的_config.yml文件
搜索关键字 deploy，按下面格式修改
deploy:
    type: git
    repo:     https://github.com/zouyingyang/zouyingyang.github.io.git （改成你的用户名)
    branch: master

然后安装一个依赖
blog$ npm install hexo-deployer-git --save

部署到github
blog$ hexo deploy
在浏览器中打开http://zouyingyang.github.io/

简写
hexo new 博客名称 == hexo n 博客名称 //新建博客
hexo generate == hexo g //生成
hexo server == hexo s //启动本地服务
hexo deploy == hexo d //部署

blog
　｜
　｜－－ _config.yml （配置文件）
　｜－－ node_modules
　｜－－ public (静态文件)
　｜－－ source (需要提交的都放在这里。比如：文章，图片)
　｜－－ db.json
　｜－－ package.json
　｜－－ scaffolds
　｜－－ themes （主题）
